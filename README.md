# tf-gcp-infra

This repository contains Terraform configurations for managing infrastructure on Google Cloud Platform (GCP).

## Pre-requisites

- Google Cloud Platform (GCP) account
- Google Cloud SDK (gcloud command-line tool)
- Terraform installed
